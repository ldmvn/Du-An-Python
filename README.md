# Dự án Website Bán Điện Thoại

## Mô tả
Website bán điện thoại di động xây dựng bằng Django.

## Chức năng
- Trang chủ (Home)
- Đăng nhập / Đăng ký
- Dashboard quản trị
- CRUD sản phẩm
- Kết nối CSDL SQLite
- Template kế thừa (header, footer)

## Công nghệ
- Python
- Django
- SQLite
- HTML, CSS

## Hướng dẫn chạy
```bash
pip install django
python manage.py migrate
python manage.py runserver
